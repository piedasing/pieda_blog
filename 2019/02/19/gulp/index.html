gulp

2019-02-19

使用到的工具、套件、技術

gulp、browser-sync、gulp-rename、SASS/SCSS、gulp-autoprefixer

目的

為求開發時，簡化重複的流程、建立一個良好的本機開發環境

步驟

  1. 安裝 node js

    https://nodejs.org/en/

  2. 安裝全域 gulp

  3. 初始化專案，產生 package.json

  4. 安裝本地所需套件

  5. 在根目錄新增 gulpfile.js

  6. 在根目錄新增 index.html 及 scss 資料夾內新增 style.scss

    gulp-demo(根目錄)
    |–scss

    |-- style.scss

    |– gulpfile.js
    |– index.html
    |– package-lock.json
    |– package.json

  7. 執行 gulp，啟動伺服器

  8. 開發網頁GO~

    當在 index.html 或是 style.scss 內存檔，瀏覽器會自動刷新
    如果要停止伺服器，則按下 ctrl + c (mac 的 ctrl 改為 cmd)
